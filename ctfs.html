Lessons & Takeaways


Every CTF teaches something new! Some key takeaways from my journey so far:

  • ✅ Always read the challenge description carefully. Sometimes, the hints are right in front of you! 👀
  • ✅ Mastering basic Linux and web security is crucial. Many challenges revolve around simple command-line tricks or common web vulnerabilities.
  • ✅ Don't be afraid to try new categories.
  • ✅ Google is your best friend. If you don’t know something, search for it! CTFs are all about learning on the go.
